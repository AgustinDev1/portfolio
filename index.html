Proyectos

POKEMON GO - PERSONAJES

Este proyecto tiene como objetivo proporcionar información detallada sobre cada personaje de Pokémon utilizando la API de PokeAPI. Los usuarios podrán explorar características, habilidades y estadísticas de sus Pokémon favoritos. Las tecnologías utilizadas en este proyecto incluyen HTML, CSS y JavaScript para la interfaz de usuario, junto con fetch API para realizar solicitudes a PokeAPI y obtener los datos necesarios.

RELOJ DIGITAL

Este proyecto consiste en un reloj digital que muestra la hora actual en Argentina. Utilizando tecnologías como HTML, CSS y JavaScript, el reloj se actualiza en tiempo real para proporcionar a los usuarios una visualización precisa de la hora local. El diseño es intuitivo y atractivo, permitiendo que los usuarios puedan consultar la hora de manera rápida y sencilla. Además, el reloj considera el uso horario de Argentina, garantizando que la información sea siempre relevante y exacta.

REPRODUCTOR DE MÚSICA

Este proyecto consiste en un reproductor de música que permite a los usuarios escuchar canciones de un artista específico. Utilizando tecnologías como HTML, CSS y JavaScript, el reproductor ofrece una interfaz amigable y fácil de usar, donde los usuarios pueden seleccionar y reproducir las pistas de su artista favorito. Además, cuenta con funcionalidades como controles de reproducción, pausa, avance y retroceso, lo que mejora la experiencia auditiva.
